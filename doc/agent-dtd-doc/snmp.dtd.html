Ultra Messaging SNMP Agent Daemon

Configuration for the Ultra Messaging SNMP Agent Daemon.



<tnwmonsnmp> Root element

The root element of the Ultra Messaging SNMP agent configuration.

NameCardinality

master-agentOne or none

@version Attribute of tnwmonsnmp

Ultra Messaging SNMP Agent DTD version used.

Possible values: 1.0 - Required

<log> Child of daemon

Indicates the file where log messages are sent.

<log>'s attributes
NameValuesDefault
typefile, syslog, consoleconsole
xml:spacedefault, preservedefault

@type Attribute of log

Type of logging. For syslog and console, the filename is ignored.

Possible values: file, syslog, console - Default value: console


@xml:space Attribute of log

How whitespace is to be handled. "default" trims leading and trailing whitespace, and compress multiple whitespace characters into a single character. "preserve" preserves the whitespace exactly as read.

Possible values: default, preserve - Default value: default

@max-age Attribute of monitor

Maximum age (in seconds) of a statistics entry with no updates before it is purged.

Default value: 60


@purge-interval Attribute of monitor

Interval (in seconds) between purge checks.

Default value: 15


@topic Attribute of monitor

Topic on which statistics are delivered.

Default value: /29west/statistics

@agentx-socket Attribute of subagent

Socket used to communicate with master agent. For Windows, this defaults to tcp:localhost:705, and for Unix/Linux this defaults to unix:/var/agentx/master.

Default value: unix:/var/agentx/master
